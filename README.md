# Hilti SLAM Challenge Submit Tool

[![PHP Composer](https://github.com/hilti-challenge/submit/actions/workflows/php.yml/badge.svg)](https://github.com/hilti-challenge/submit/actions/workflows/php.yml)
[![Node.js Encore](https://github.com/hilti-challenge/submit/actions/workflows/node.js.yml/badge.svg)](https://github.com/hilti-challenge/submit/actions/workflows/node.js.yml)

This tool enables researches to submit their solution to the Hilti SLAM challenge. The solution is evaluated and the result is shown to the researcher. 
If the deadline has passed, the best result of each user appears on the leaderboard.

The symfony/PHP web application provides the user-facing functionality. AWS is used to evaluate the submissions.
