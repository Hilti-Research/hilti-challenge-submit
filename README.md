# Hilti SLAM Challenge Submit Tool

[![PHP Composer](https://github.com/hilti-challenge/hilti-challenge-submit/actions/workflows/php.yml/badge.svg)](https://github.com/hilti-challenge/hilti-challenge-submit/actions/workflows/php.yml)
[![Node.js Encore](https://github.com/hilti-challenge/hilti-challenge-submit/actions/workflows/node.js.yml/badge.svg)](https://github.com/hilti-challenge/hilti-challenge-submit/actions/workflows/node.js.yml)

This tool enables participants of the SLAM challenge to submit their solution. The solution is evaluated and the result is shown to the user. Further, a leaderboard shows the best submissions.

The symfony/PHP web application provides the user-facing functionality. AWS is used to evaluate the submissions.
